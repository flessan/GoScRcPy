# scrcpy (v3.3.4)

<img src="https://raw.githubusercontent.com/Genymobile/scrcpy/master/app/data/icon.svg" width="128" height="128" alt="scrcpy" align="right" />

_pronounced "**scr**een **c**o**py**"_

This application mirrors Android devices (video and audio) connected via USB or
[TCP/IP](doc/connection.md#tcpip-wireless) and allows control using the
computer's keyboard and mouse. It does not require _root_ access or an app
installed on the device. It works on _Linux_, _Windows_, and _macOS_.

## Must-know tips

 - [Reducing resolution](https://github.com/Genymobile/scrcpy/blob/master/README.md#reducing-resolution) may greatly improve performance
   (`scrcpy -m1024`)
 - [_Right-click_](https://github.com/Genymobile/scrcpy/blob/master/README.md#mouse-bindings) triggers `BACK`
 - [_Middle-click_](https://github.com/Genymobile/scrcpy/blob/master/README.md#mouse-bindings) triggers `HOME`
 - <kbd>Alt</kbd>+<kbd>f</kbd> toggles [fullscreen](https://github.com/Genymobile/scrcpy/blob/master/README.md#fullscreen)
 - There are many other [shortcuts](https://github.com/Genymobile/scrcpy/blob/master/README.md#shortcuts)
